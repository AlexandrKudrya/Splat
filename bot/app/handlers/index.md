# bot/app/handlers/

aiogram роутеры — обработчики команд и сообщений Telegram.

## Что здесь

| Файл | Описание |
|---|---|
| `start.py` | Обработчик команды `/start` — авторизация на backend, приветствие |
| `order.py` | FSM-флоу заказа: `/new`, загрузка фото, `/add @username` |
| `notifications.py` | Функции для отправки уведомлений: чек распарсен, нужно оплатить, все оплатили |

## Правило

Хендлеры не содержат бизнес-логики. Они вызывают `client/api.py` для получения данных и формируют сообщения.
